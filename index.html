Setup Requirements

  • dotnet core (2.+)
  • yarn and node
  • fake command line tool

Installing Fake

dotnet tool install fake-cli -g

  • Linux/OSx: ~/.dotnet/tools
  • Windows: %USERPROFILE%\.dotnet\tools

Installing and Running SAFE

  • Command Line
    • dotnet new -i SAFE.Template
    • dotnet new SAFE
    • fake build --target run
